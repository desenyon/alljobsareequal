# AllJobsAreEqual

A minimalist, inclusive hiring portal for fair-chance and disabled candidates. Built for hackathon speed with Next.js 14 (App Router), TypeScript, Tailwind, and shadcn-style components.

## Notes

- Privacy defaults are OFF; candidates toggle sharing per application.
- Seed data can be swapped for a real DB later; store merges localStorage updates.
- Admin route is hardcoded; hide jobs to remove from public list.
